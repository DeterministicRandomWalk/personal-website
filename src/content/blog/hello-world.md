This is the first post on the site.

The goal is simple: keep a personal place for blog posts, technical notes, and ideas that are useful enough to write down. The site starts small on purpose, using Astro for a fast static blog and preparing for deployment on Cloudflare.

Future posts can live in this folder as Markdown or MDX files. Each post needs a title, description, publication date, and optional hero image in the frontmatter.
